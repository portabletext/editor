Portable Text is a JSON format — to display it, you need a **serializer** that converts the structured data into your target output. Official serializers are available for most popular frameworks, with community packages covering the rest.

:::note[Prop name differences across frameworks]
Most renderers use `types` and `marks` (plural) for custom components. **Svelte and Astro use `type` and `mark` (singular).** Check the examples below for your framework.
:::

## Quick Start

The fastest way to render Portable Text depends on your stack:

<Tabs>
  <TabItem label="React">
    ```bash
    npm install @portabletext/react
    ```

    ```tsx
    import {PortableText} from '@portabletext/react'

    function Article({content}) {
      return <PortableText value={content} />
    }
    ```

    That's it for basic rendering. The component renders sensible HTML defaults — paragraphs, headings, bold, italic, links, and lists all work out of the box.

    To customize how specific types render, pass a `components` prop:

    ```tsx
    import {PortableText} from '@portabletext/react'

    const components = {
      types: {
        image: ({value}) => <img src={value.url} alt={value.alt} />,
      },
      marks: {
        link: ({children, value}) => (
          <a href={value.href} target="_blank" rel="noopener">
            {children}
          </a>
        ),
      },
    }

    function Article({content}) {
      return <PortableText value={content} components={components} />
    }
    ```

    [Full documentation →](https://github.com/portabletext/react-portabletext/)

  </TabItem>

  <TabItem label="HTML">
    ```bash
    npm install @portabletext/to-html
    ```

    ```ts
    import {toHTML} from '@portabletext/to-html'

    const html = toHTML(portableTextContent)
    ```

    Customize rendering with the `components` option. Use `uriLooksSafe()` to prevent `javascript:` URI injection in links:

    ```ts
    import {toHTML, uriLooksSafe} from '@portabletext/to-html'

    const html = toHTML(portableTextContent, {
      components: {
        types: {
          image: ({value}) => `<img src="${value.url}" alt="${value.alt}" />`,
        },
        marks: {
          link: ({children, value}) => {
            const href = value.href || ''
            if (uriLooksSafe(href)) {
              const rel = href.startsWith('/') ? undefined : 'noreferrer noopener'
              return `<a href="${href}" rel="${rel}">${children}</a>`
            }
            return children
          },
        },
      },
    })
    ```

    [Full documentation →](https://github.com/portabletext/to-html/)

  </TabItem>

  <TabItem label="Vue">
    ```bash
    npm install @portabletext/vue
    ```

    ```vue
    <script setup>
    import {PortableText} from '@portabletext/vue'

    defineProps(['value'])

    <template>
      <PortableText :value="value" />
    </template>
    ```

    [Full documentation →](https://github.com/portabletext/vue-portabletext/)

  </TabItem>

  <TabItem label="Svelte">
    ```bash
    npm install @portabletext/svelte -D
    ```

    ```svelte
    <script>
      import {PortableText} from '@portabletext/svelte'

      let {value} = $props()
    </script>

    <PortableText value={value} />
    ```

    Requires Svelte 5+.

    [Full documentation →](https://github.com/portabletext/svelte-portabletext/)
